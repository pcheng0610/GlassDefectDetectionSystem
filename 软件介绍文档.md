# 🏭 玻璃缺陷检测系统 - 软件介绍

## 📖 项目概述

**玻璃缺陷检测系统** 是一款基于深度学习的智能工业检测软件，专为玻璃制品生产企业设计。该系统采用先进的 YOLOv8 目标检测算法，能够自动识别和定位玻璃产品表面的各类缺陷，帮助企业提高产品质量检测效率，降低人工成本。

---

## 🎯 核心功能

### 1️⃣ 实时缺陷检测
- **在线检测模式**：实时上传玻璃图片，即时获取检测结果
- **离线检测模式**：批量处理本地图片，支持大规模数据检测
- **多缺陷识别**：可同时检测划痕、气泡、裂纹、杂质等多种缺陷类型
- **精确定位**：使用边界框准确标注缺陷位置

### 2️⃣ 批量处理
- **批量上传**：支持一次上传多张图片进行批量分析
- **自动处理**：后台自动队列处理，无需等待
- **结果导出**：检测结果可导出为图片格式，便于存档和分享

### 3️⃣ 历史记录管理
- **检测记录**：自动保存所有历史检测记录
- **结果查询**：支持按时间、缺陷类型等条件查询历史记录
- **数据追溯**：完整保留检测图片、结果、时间戳等信息

### 4️⃣ 数据可视化
- **统计分析**：生成缺陷类型分布、检测数量统计等图表
- **趋势分析**：展示质量变化趋势，辅助生产决策
- **实时看板**：Dashboard 实时显示检测状态和关键指标

### 5️⃣ 用户管理
- **账户系统**：支持多用户注册和登录
- **权限管理**：不同用户角色拥有不同操作权限
- **个性化配置**：支持用户自定义检测参数

## 📊 检测能力

### 支持的缺陷类型
1. **划痕 (Scratch)** - 玻璃表面线性损伤
2. **气泡 (Bubble)** - 玻璃内部气体夹杂
3. **裂纹 (Crack)** - 玻璃结构性裂缝
4. **杂质 (Inclusion)** - 异物夹杂
5. **斑点 (Spot)** - 表面污染或腐蚀
6. **其他缺陷** - 支持自定义扩展

### 性能指标
- **检测速度**: < 500ms/张 (取决于硬件配置)
- **检测精度**: mAP > 95% (在测试集上)
- **最小缺陷**: 可检测 0.1mm² 级别缺陷
- **并发处理**: 支持多进程并行推理

## 🚀 部署方式

### 本地部署
1. 下载项目代码
2. 安装 Python 依赖：`pip install -r requirements.txt`
3. 启动服务器：`python app.py`
4. 浏览器访问：`http://localhost:5000`

### Docker 部署 (可选)
```bash
# 构建镜像
docker build -t glass-detection .

# 运行容器
docker run -p 5000:5000 glass-detection
```

### 打包成安装包
- 使用 Electron + PyInstaller 打包成 Windows 可执行文件
- 一键安装，离线使用
- 包含所有依赖和模型文件

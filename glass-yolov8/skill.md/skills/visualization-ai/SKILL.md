# 可视化AI助手

这是一个专门用于玻璃缺陷检测系统的AI助手，提供数据分析和智能建议功能。

## 功能说明

### 1. 数据分析
- 分析检测数据统计
- 生成检测报告
- 统计各类别分布
- 分析置信度分布
- 时间趋势分析

### 2. 智能建议
- 根据检测数据提供优化建议
- 针对常见问题给出解决方案
- 提供质量改进建议

## 使用场景

当用户需要：
- 查看检测数据统计报告
- 了解各类别检测分布
- 获取优化建议
- 分析检测质量趋势

## 数据分析功能

### 分析内容
- **总检测次数**: 统计所有检测记录
- **瑕疵数量**: 统计各类缺陷数量
- **正常数量**: 未检测到瑕疵的记录数
- **瑕疵率**: 计算缺陷占比
- **类别分布**: 各类缺陷的分布情况
- **置信度分布**: 不同置信度区间的统计
- **时间趋势**: 按小时的检测趋势

### 输出格式
```
📊 **数据分析报告**

- 总检测次数：XX 次
- 瑕疵数量：XX 个
- 正常数量：XX 个
- 瑕疵率：XX.X%

**类别分布：**
- 黑点：XX 个
- 划痕：XX 个
- 坑点：XX 个
...
```

## 智能建议功能

### 分析维度
- **缺陷类型分析**: 识别主要缺陷类型
- **置信度分析**: 评估检测可靠性
- **时间趋势分析**: 发现异常时间段
- **分布规律分析**: 找出规律和问题点

### 输出格式
```
🤖 **智能建议**

1. **主要问题**: 分析主要缺陷类型
2. **改进建议**: 针对性优化建议
3. **注意事项**: 检测和生产建议
```

## 使用示例

用户可以直接询问：
- "生成数据分析报告"
- "给我智能建议"
- "分析一下检测数据"
- "有什么优化建议"
- "帮我看看检测结果"

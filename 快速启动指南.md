# 🚀 快速启动指南

## 项目说明
基于 Flask + YOLOv8 的玻璃缺陷检测系统

---

## ⚡ 一键启动（推荐）

### Windows 系统

**方法 1：双击启动脚本**
```bash
cd glass-yolov8
python app.py
```

**方法 2：使用批处理文件**
```bash
# 创建 start.bat 文件，双击即可启动
@echo off
cd /d %~dp0\glass-yolov8
python app.py
pause
```

---

## 📝 详细启动步骤

### 步骤 1：打开终端/命令行
- **Windows**: 按 `Win + R`，输入 `cmd`，回车
- 或者在文件夹空白处，按住 `Shift` + 右键，选择"在此处打开 PowerShell 窗口"

### 步骤 2：进入项目目录
```bash
cd e:\zjzjzj\glass-yolov8
```

### 步骤 3：启动后端服务器
```bash
python app.py
```

### 步骤 4：打开浏览器访问
在浏览器地址栏输入：
```
http://127.0.0.1:5002
```
或
```
http://localhost:5002
```

## ⚠️ 常见问题

### Q1: 端口被占用
**错误信息**: `Address already in use`
**解决方法**: 
```bash
# 修改端口号，在 app.py 中更改 port 参数
app.run(port=5002)
```

### Q2: 模型文件找不到
**解决方法**: 确保 `runs/detect/yolov8m_glass_detection/weights/best.pt` 文件存在

### Q3: 权限错误
**错误信息**: `Permission denied`
**解决方法**: 以管理员身份运行命令行

---如需在其他服务器部署本系统，请按以下步骤操作：

1. 复制模型文件 ：将 runs/detect/yolov8m_glass_detection/weights/best.pt 复制到新服务器的相同路径
2. 修改模型路径 ：在 pythonWeb/views/detection.py 文件第13行，找到 MODEL_PATH 变量，将其修改为您的模型文件路径：MODEL_PATH = r"您的模型路径\best.pt"
